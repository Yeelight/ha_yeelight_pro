# 其他讨论项

- 单组件带不带index的讨论

    - 目前单组件不支持index，而空调网关即使只接了一路也要携带index

        - 方案：配置协议不调整；控制协议根据产品定义的最大组件数来判断，如果最大组件数为1，那么将子设备去除，信息放置在设备维度。该方案规避了单组件设备也必须支持1\-xxx的问题。
