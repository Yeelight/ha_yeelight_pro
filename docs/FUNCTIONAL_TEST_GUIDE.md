# 实际功能测试指南

## 测试环境要求

- Home Assistant 2024.1.0 或更高版本
- Python 3.11 或更高版本
- Yeelight Pro 设备（云端或私有部署）

## 测试步骤

### 1. 安装集成

#### 方法 A: 本地开发安装

```bash
# 1. 克隆仓库
git clone https://github.com/Yeelight/ha_yeelight_pro.git
cd ha_yeelight_pro

# 2. 复制到 HA 配置目录
cp -r custom_components/yeelight_pro /path/to/homeassistant/config/custom_components/

# 3. 重启 Home Assistant
```

#### 方法 B: HACS 安装（发布后）

1. 打开 HACS
2. 搜索 "Yeelight Pro"
3. 点击安装
4. 重启 Home Assistant

### 2. 配置集成

1. 打开 Home Assistant
2. 进入 设置 → 设备与服务 → 添加集成
3. 搜索 "Yeelight Pro"
4. 选择连接模式：
   - **云端模式**: 选择 "Yeelight Pro 云端"
   - **私有部署**: 选择 "私有部署（Lucore）"

#### 云端模式配置

1. 输入 Access Token
2. 点击提交
3. 选择家庭
4. 完成配置

#### 私有部署配置

1. 输入服务器地址（例如：192.168.1.100:8080）
2. 输入 Access Token
3. 输入家庭 ID
4. 点击提交
